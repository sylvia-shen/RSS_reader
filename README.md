# RSS Reader
This a rss reader ruby script to do some live coding challenge

## Description
Start at: 00:21
End at: 01:20

## How to use?
#### Install the gem
```
bundle install
```

#### Execute the script directly
```
ruby ./read_rss 'https://rss-feed-url'
```
